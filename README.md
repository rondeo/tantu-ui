# tantu-ui



## 快速开始

1. `npm run i`
2. `npm run start`

## 目录结构


```html
/tantu-ui
    |-- build/
    |-- demo/ Demo程序
    |-- dist/
    |-- packages/ 在这里写各种组件
    |-- src/ 打包入口
    |-- types/
    |-- typings/
    |-- .eslintrc.json
    |-- package.json
```
